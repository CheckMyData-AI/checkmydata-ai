### T2 — SQL-dialect `distinct_values` / `approx_stats` (base default + PG/MySQL/CH overrides)  [G4a, dep W0 C-D]  `DBIDX-D2/D9 plumbing`

**Files:** `backend/app/connectors/base.py` (base-class SQL defaults — **only** the two new methods, do
not touch W0-owned dataclasses), `backend/app/connectors/postgres.py`, `mysql.py`, `clickhouse.py`,
`backend/tests/unit/test_connectors_extended.py`

**Interfaces (produces `list[str]` / `ColumnStats`, dialect-correct quoting):**
The base `DatabaseAdapter` gets SQL defaults that build a parametrized-safe identifier-quoted query
and run through `self.execute_query`. Reuse `self._quote_identifier` (already dialect-aware for
`mysql` vs default). ClickHouse overrides `approx_stats` to use `uniqExact`/`min`/`max` and
`countIf(x IS NULL)`; base default uses `COUNT(DISTINCT col)`.

```python
# DatabaseAdapter (base) — SQL default
async def distinct_values(self, table_name: str, column: str, limit: int = 50) -> list[str]:
    tq, cq = self._quote_identifier(table_name), self._quote_identifier(column)
    qr = await self.execute_query(
        f"SELECT DISTINCT {cq} FROM {tq} WHERE {cq} IS NOT NULL ORDER BY {cq} LIMIT {int(limit)}"
    )
    if qr.error or not qr.rows:
        return []
    return [str(r[0]) for r in qr.rows if r[0] is not None][:limit]

async def approx_stats(self, table_name: str, column: str) -> ColumnStats:
    tq, cq = self._quote_identifier(table_name), self._quote_identifier(column)
    qr = await self.execute_query(
        f"SELECT COUNT(DISTINCT {cq}) AS dc, "
        f"SUM(CASE WHEN {cq} IS NULL THEN 1 ELSE 0 END) AS nulls, COUNT(*) AS total, "
        f"MIN({cq}) AS mn, MAX({cq}) AS mx FROM {tq}"
    )
    if qr.error or not qr.rows:
        return ColumnStats()
    dc, nulls, total, mn, mx = qr.rows[0]
    null_rate = (nulls / total) if total else None
    return ColumnStats(distinct_count=dc, null_rate=null_rate, min_value=mn, max_value=mx)
```
