### Task 1: CODEIDX-C4 — incremental merge re-resolves cross-file CALLS/IMPORTS  [depends:W0-C4-CHAR]

**Why (audit §3):** `save_incremental` (`code_graph_service.py:104-174`) keeps unchanged-file edges verbatim and splices in the changed subset. A **new** cross-file call from an *unchanged* caller to a symbol in a *changed* callee never gets a CALLS edge, and a caller edge to a renamed callee is only *pruned* (C4 compounded by C7). The graph drifts while `graph_build` reports "completed". Fix: on incremental runs, expand the parse set to the **reverse-dependency set** of the changed files (files that import a changed file, plus the changed files themselves), re-parse them, and rebuild edges for that closure so cross-file CALLS/IMPORTS are recomputed against current symbols.

**Files:**
- Modify: `backend/app/knowledge/pipeline_runner.py` — `_collect_files_for_ast` (~1330-1369) and the incremental branch of `_run_graph_build` (~1505-1544): compute the reverse-dependency closure and include it in `affected_files` + the parse set.
- Modify: `backend/app/knowledge/code_graph.py` — add `CodeGraphBuilder.reverse_dependents(existing: CodeGraph, changed_files: set[str]) -> set[str]` (static) that returns files whose IMPORTS edges (or import refs) point into any changed file.
- Test: `backend/tests/unit/test_code_graph_reverse_deps.py` (new, unit on the helper).
- Test: `backend/tests/integration/test_code_graph_incremental_cross_file.py` (consume/flip the W0 characterization test).

**Interfaces:**
- Consumes (from W0): `test_unchanged_caller_relinks_after_incremental_change` — the failing characterization asserting the *current* drift. Task 1 flips it to green.
- Consumes: `CodeGraphService.load_graph(session, project_id) -> CodeGraph | None`, `CodeGraph.symbols: dict[str, Symbol]`, `CodeGraph.edges: list[GraphEdge]`, `GraphEdge(src_uid, dst_uid, edge_type, confidence, attrs)`, `EDGE_IMPORTS`.
- Produces: `CodeGraphBuilder.reverse_dependents(existing: CodeGraph, changed_files: set[str]) -> set[str]` (repo-relative paths, excluding the changed files themselves). Consumed by T7 (ignore set must not exclude a reverse-dependent) and T10.

- [ ] **Step 3: Implement `reverse_dependents` in `code_graph.py`**

Add to `class CodeGraphBuilder` (after `_break_inheritance_cycles`):

```python
    @staticmethod
    def reverse_dependents(existing: CodeGraph, changed_files: set[str]) -> set[str]:
        """Files that import a symbol defined in any ``changed_files`` path.

        Used on incremental runs so a *new* cross-file CALLS/IMPORTS edge from
        an unchanged caller is recomputed against the changed callee's current
        symbols (CODEIDX-C4). Excludes the changed files themselves.
        """
        if not changed_files:
            return set()
        sym_file = {uid: sym.file_path for uid, sym in existing.symbols.items()}
        dependents: set[str] = set()
        for edge in existing.edges:
            if edge.edge_type != EDGE_IMPORTS:
                continue
            dst_file = sym_file.get(edge.dst_uid)
            if dst_file is None or dst_file not in changed_files:
                continue
            if edge.src_uid.startswith("file:"):
                src_file = edge.src_uid[len("file:") :]
            else:
                src_file = sym_file.get(edge.src_uid)
            if src_file and src_file not in changed_files:
                dependents.add(src_file)
        return dependents
```

```python
async def test_unchanged_caller_relinks_after_incremental_change(
    db_session: AsyncSession, project_id: str
```

### Task 2: CODEIDX-C7 — drop `line` from symbol UID identity  [depends:T1]

**Why (audit §3):** `_make_uid` (`ast_parser.py:361-363`) includes `start_line`, so inserting a line above an untouched symbol changes its UID; the merge treats it as a new symbol and drops inbound edges (compounds C4). `start_line` is already a first-class `Symbol` attribute and a `code_graph_symbols.start_line` column, and nothing parses the line back out of the UID (verified: no `uid.split(':')` for line anywhere). So the line is safe to drop from identity and keep only as an attribute.

```python
def _extract_bases(node, grammar: LanguageGrammar, source: bytes) -> tuple[str, ...]:
    """Collect base-class / implemented-interface identifiers from AST heritage.

    Handles Python ``argument_list`` superclasses, TS/JS ``class_heritage``
    (extends + implements clauses), and Java/C# ``superclass``/
    ``base_list``/``super_interfaces``. Best-effort: unknown grammars yield ().
    """
    bases: list[str] = []

    def _idents(n) -> None:
        stack = [n]
        while stack:
            cur = stack.pop()
            if cur.type in ("identifier", "type_identifier"):
                txt = cur.text.decode("utf-8", "replace") if cur.text else ""
                if txt and txt not in ("object",):
                    bases.append(txt)
                continue
            stack.extend(cur.children)

    for child in node.children:
        if child.type in (
            "superclasses",            # python: class C(A, B)
            "class_heritage",          # ts/js: extends X implements Y
            "extends_clause",
            "implements_clause",
            "superclass",              # java
            "super_interfaces",
            "base_list",               # c#
        ):
            _idents(child)
    # De-dupe preserving order.
    seen: set[str] = set()
    out: list[str] = []
    for b in bases:
        if b not in seen:
            seen.add(b)
            out.append(b)
    return tuple(out)
```

```python
            for pf in parsed_files.values():
                for call in pf.call_sites:
                    degree[call.caller_uid] += 1
                    degree[call.callee_name] += 1  # name-level fan-in hint
            def _score(s: Symbol) -> tuple[int, int]:
                by_name = degree.get(s.name, 0)
                by_uid = degree.get(s.uid, 0)
                public = 0 if s.name.startswith("_") else 1
                return (by_uid + by_name + public, public)
            ranked = sorted(all_symbols, key=_score, reverse=True)
            kept = ranked[: self._max_symbols]
            logger.warning(
                "code_graph: symbol cap %d exceeded (%d), pruned by degree to %d",
                self._max_symbols,
                len(all_symbols),
                len(kept),
            )
            all_symbols = kept
```

```python
def _resolve_enum_to_columns(knowledge: ProjectKnowledge) -> None:
    """Attach enum value-sets to columns by DECLARED TYPE / FK, not fuzzy substring.

    Precedence (CODEIDX-C15):
      1. Column ``col_type`` names the enum exactly (case-insensitive) — the
         common ``status: Mapped[OrderStatus]`` / ``@Column(enum=OrderStatus)`` case.
      2. Column ``fk_target`` resolves to the enum name.
      3. Column NAME equals the enum name minus a trailing "Enum"/"Type" suffix
         (conservative exact match), else leave ``enum_values`` empty.
    """
    enum_by_key: dict[str, list[str]] = {}
    for enum_def in knowledge.enums:
        enum_by_key[enum_def.name.lower()] = enum_def.values
    for entity in knowledge.entities.values():
        for col in entity.columns:
            if col.enum_values:
                continue
            ctype = (col.col_type or "").lower()
            fk = (col.fk_target or "").lower().split(".")[-1]
            cname = col.name.lower()
            match: list[str] | None = None
            for ename, values in enum_by_key.items():
                stripped = ename
                for suffix in ("enum", "type", "status", "choices"):
                    if stripped.endswith(suffix) and stripped != suffix:
                        stripped = stripped[: -len(suffix)]
                if ename in ctype or fk == ename or cname == ename or cname == stripped:
                    match = values
                    break
            if match is not None:
                col.enum_values = match
```

```python
_IRREGULAR_PLURALS = {
    "person": "people", "child": "children", "man": "men", "woman": "women",
    "foot": "feet", "tooth": "teeth", "mouse": "mice", "goose": "geese",
    "datum": "data", "index": "indices", "matrix": "matrices",
}


def _pluralize(word: str) -> str:
    """Dependency-free English pluralizer for table-name inference (CODEIDX-C16)."""
    lower = word.lower()
    if lower in _IRREGULAR_PLURALS:
        return _IRREGULAR_PLURALS[lower]
    if lower.endswith("y") and len(lower) > 1 and lower[-2] not in "aeiou":
        return word[:-1] + "ies"
    if lower.endswith(("s", "x", "z", "ch", "sh")):
        return word + "es"
    return word + "s"


def _model_name_to_table(model_name: str) -> str:
    """Convert CamelCase model name to snake_case pluralized table name."""
    s1 = re.sub(r"(.)([A-Z][a-z]+)", r"\1_\2", model_name)
    snake = re.sub(r"([a-z0-9])([A-Z])", r"\1_\2", s1).lower()
    # Pluralize only the last path segment (e.g. user_profile -> user_profiles).
    head, _, tail = snake.rpartition("_")
    plural_tail = _pluralize(tail)
    return f"{head}_{plural_tail}" if head else plural_tail


def _model_to_table(
    model_name: str,
    knowledge: ProjectKnowledge,
    live_tables: set[str] | None = None,
) -> str:
    """Model -> table name via cache, live-table cross-check, then heuristic."""
    entity = knowledge.entities.get(model_name)
    if entity and entity.table_name:
        return entity.table_name
    plural = _model_name_to_table(model_name)
    if live_tables:
        snake = re.sub(r"([a-z0-9])([A-Z])", r"\1_\2",
                       re.sub(r"(.)([A-Z][a-z]+)", r"\1_\2", model_name)).lower()
        for cand in (plural, snake, model_name.lower()):
            if cand in live_tables:
                return cand
    return plural
```

```python
# backend/app/eval/graph_benchmark.py
"""Graph-quality benchmark gating the code_graph/lineage flag flip (W6, spec §9).

Builds the code graph from a tiny known fixture repo and asserts minimum
symbol / CALLS / EXTENDS / IMPORTS counts. Deterministic (no LLM, no network).
Run as ``python -m app.eval.graph_benchmark`` — exits non-zero on FAIL so it
can be a CI/rollout gate before flipping the flags.
"""

from __future__ import annotations

import sys
import tempfile
from dataclasses import dataclass, field
from pathlib import Path

from app.knowledge.ast_parser import ASTParser
from app.knowledge.code_graph import (
    EDGE_CALLS, EDGE_EXTENDS, EDGE_IMPORTS, CodeGraphBuilder,
)
from app.knowledge.shared_ignore import is_ignored_path

_MIN_SYMBOLS = 7
_MIN_CALLS = 1
_MIN_EXTENDS = 1
_MIN_IMPORTS = 1


@dataclass
class GraphBenchmarkResult:
    symbols: int
    calls: int
    extends: int
    imports: int
    passed: bool
    failures: list[str] = field(default_factory=list)


def _fixture(root: Path) -> None:
    # Imported lazily to reuse the single source of truth from the test module.
    from tests.integration.test_code_graph_end_to_end import _write_fixture_repo

    _write_fixture_repo(root)


def run_graph_benchmark(*, repo_root: Path | None = None) -> GraphBenchmarkResult:
    tmp: tempfile.TemporaryDirectory | None = None
    if repo_root is None:
        tmp = tempfile.TemporaryDirectory()
        repo_root = Path(tmp.name)
        _fixture(repo_root)
    try:
        parser = ASTParser()
        parsed = {}
        for path in repo_root.rglob("*"):
            if not path.is_file():
                continue
            rel = str(path.relative_to(repo_root)).replace("\\", "/")
            if is_ignored_path(rel):
                continue
            pf = parser.parse_file(repo_root, rel)
            if pf is not None:
                parsed[rel] = pf
        graph = CodeGraphBuilder().build(parsed)
        n_sym = len(graph.symbols)
        n_calls = sum(1 for e in graph.edges if e.edge_type == EDGE_CALLS)
        n_ext = sum(1 for e in graph.edges if e.edge_type == EDGE_EXTENDS)
        n_imp = sum(1 for e in graph.edges if e.edge_type == EDGE_IMPORTS)
        failures: list[str] = []
        if n_sym < _MIN_SYMBOLS:
            failures.append(f"symbols {n_sym} < {_MIN_SYMBOLS}")
        if n_calls < _MIN_CALLS:
            failures.append(f"CALLS {n_calls} < {_MIN_CALLS}")
        if n_ext < _MIN_EXTENDS:
            failures.append(f"EXTENDS {n_ext} < {_MIN_EXTENDS}")
        if n_imp < _MIN_IMPORTS:
            failures.append(f"IMPORTS {n_imp} < {_MIN_IMPORTS}")
        return GraphBenchmarkResult(
            symbols=n_sym, calls=n_calls, extends=n_ext, imports=n_imp,
            passed=not failures, failures=failures,
        )
    finally:
        if tmp is not None:
            tmp.cleanup()


def main() -> int:
    res = run_graph_benchmark()
    status = "PASS" if res.passed else "FAIL"
    print(
        f"graph_benchmark: {status} "
        f"symbols={res.symbols} CALLS={res.calls} EXTENDS={res.extends} IMPORTS={res.imports}"
    )
    for f in res.failures:
        print(f"  - {f}")
    return 0 if res.passed else 1


if __name__ == "__main__":
    sys.exit(main())
```
